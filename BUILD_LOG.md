# EstateFlow – Build Log

## Project Overview

EstateFlow is a full-stack Real Estate Listings Portal developed using the MERN ecosystem (Next.js, Express.js, Prisma, and PostgreSQL). The goal of the project is to help users browse properties and allow real estate agents to manage customer inquiries through a lead dashboard with AI-inspired lead scoring.

This project was built as part of the DigitalSofts technical evaluation.

---

# Features Implemented

### Authentication

* User Signup
* User Login
* JWT-based Authentication
* Protected Backend Routes

### Property Management

* Add Property
* View All Properties
* Search and Filter Properties
* Property Details Page
* Delete Property

### Inquiry Management

* Inquiry Form for each Property
* Save Inquiries in PostgreSQL
* Delete Inquiry
* Agent Dashboard for Viewing Inquiries

### Lead Scoring

A simple lead scoring system was implemented to categorize customer inquiries into:

* High
* Medium
* Low

The score is automatically generated when an inquiry is created and displayed in the dashboard.
